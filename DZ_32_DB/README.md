## DB Stats Tool  

Консольное приложение на языке C для работы с базой данных PostgreSQL. Позволяет автоматически создавать таблицу, 
загружать в неё данные из CSV-файла и вычислять статистические параметры по заданной числовой колонке.  

## Функционал  

* Автоматизация: создание таблицы (если она отсутствует) и загрузка данных через протокол COPY.  
* Статистика: расчет среднего значения, максимума, минимума, дисперсии и суммы.  
* Безопасность: проверка типов данных (запрет расчетов для нечисловых колонок).  
* Обработка ошибок: детальный вывод при проблемах с подключением или SQL-запросами.  

## Требования  
Для сборки и запуска в среде WSL2 (Ubuntu) необходимы:  

* Компилятор gcc и утилита make.  
* Установленный клиент и заголовочные файлы PostgreSQL:  
```sh
$ sudo apt install libpq-dev postgresql-client
```
* Запущенный сервер PostgreSQL.  

## Сборка  

Используйте Makefile для сборки проекта:  

* make — сборка основной версии в папку build/.  
* make debug — сборка отладочной версии с флагом -g.  
* make clean — удаление временных файлов и бинарников.  

## Использование  

Запуск программы осуществляется с передачей аргументов:  
```sh
$ ./build/db_postgres <имя_бд> <таблица> <колонка> [путь_к_csv]
```

## Пример автоматического теста  

Для быстрой проверки работоспособности выполните:  
```sh
$ make test
```
Эта команда:  

   1. Удалит старую таблицу (сброс данных).  
   2. Сгенерирует тестовый файл oscars.csv.  
   3. Скомпилирует программу.  
   4. Загрузит данные и выведет статистику по колонке age.  

Вывод:  
```sh
$ make test
mkdir -p build
gcc -Wall -Wextra -Wpedantic -std=c11 -I/usr/include/postgresql -o build/db_postgres src/db_postgres.c -L/usr/lib/x86_64-linux-gnu -lpq
Dropping table oscar in database postgres...
DROP TABLE
Test data generated in oscars.csv
./build/db_postgres postgres oscar age oscars.csv
Successfully loaded data from 'oscars.csv' into table 'oscar'.

--- Statistics for oscar.age ---
Average:  34.5
Maximum:  62
Minimum:  22
Variance: 202.3
Sum:      207
------------------------------
```

## Настройка доступа  

Программа использует стандартные параметры подключения к localhost. 
Пароль для автоматизации тестов задается в Makefile через переменную PGPASSWORD.
