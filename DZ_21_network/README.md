# Telehack ASCII Art Client (C11)  

Простая утилита на языке Си для преобразования текста в ASCII-арт с использованием API сервиса [telehack.com](https://telehack.com). 
Программа работает через протокол Telnet (порт 23) без использования сторонних библиотек.  

## Особенности  

*   **Чистый Си**: Только POSIX sockets (никаких `libcurl` или `telnet-client`).  
*   **Обработка аргументов**: Поддерживает склейку нескольких слов без кавычек.  
*   **Сетевая устойчивость**: Использование `poll()` для обработки таймаутов и сетевых задержек.  
*   **Фильтрация трафика**: Автоматическая очистка вывода от управляющих символов Telnet (IAC) и лишних символов переноса строки.  
*   **Синхронизация**: Дожидается готовности сервера перед отправкой команд, что гарантирует чистый вывод.  

## Требования  

*   Компилятор `gcc` (поддержка стандарта C11).  
*   Система семейства Unix (Linux, macOS).  
*   Доступ в интернет (порт 23 TCP).  

## Сборка и установка  

Сборка осуществляется с помощью прилагаемого `Makefile`:  

```sh
# Сборка проекта
make

# Запуск тестов (проверка связи и популярных шрифтов)
make test
```

## Использование  

```sh
./ascii_art <шрифт> <ваш текст...>
```

## Примеры:  

```sh
./ascii_art shadow Hello World
./ascii_art slant Network Programming
./ascii_art bubble Clean Code
```
