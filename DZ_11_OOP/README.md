# GtkTreeView File Browser

Простое графическое приложение на языке C с использованием библиотеки GTK+ 3, предназначенное для рекурсивного отображения файлов и директорий.

## Установка зависимостей  

Перед сборкой убедитесь, что в вашей системе установлены заголовочные файлы GTK3 и инструменты сборки.  

### Ubuntu / Debian / Linux Mint
```bash
sudo apt update
sudo apt install build-essential libgtk-3-dev pkg-config
```

## Функциональные возможности

*   **Древовидное отображение**: Рекурсивный обход текущего каталога и отображение вложенных папок.
*   **Навигация**: 
    *   Двойной клик по папке устанавливает её как новый корень (вход "вглубь").
    *   Кнопка "На уровень выше" позволяет вернуться в родительский каталог.
*   **Метаданные**: Отображение системных иконок и человекочитаемого размера файлов (КБ, МБ, ГБ).
*   **Сортировка**: Возможность сортировки элементов по имени и реальному размеру (в байтах) при клике на заголовки колонок.
*   **Строка состояния**: Отображение общего количества объектов в текущем просматриваемом корне.
*   **Управление**: Быстрый выход из приложения по нажатию клавиши `Esc`.

## Требования

Для сборки и запуска необходимы:
*   Компилятор `gcc`
*   Библиотека `GTK+ 3.0` (пакет `libgtk-3-dev` в Debian/Ubuntu)
*   Утилита `pkg-config`

## Сборка

Сборка осуществляется с помощью `Makefile`. Используются строгие флаги компиляции: `-Wall -Wextra -Wpedantic -std=c11`.

```sh
make
```

## Запуск  
После компиляции запустите исполняемый файл:
```sh
./app_gtk
```

## Очистка  
```sh
make clean
```

## Управление  
* Двойной клик / Enter: Открыть папку (сменить корень).  
* Кнопка сверху: Перейти в родительскую директорию.  
* Esc: Мгновенный выход из приложения.
