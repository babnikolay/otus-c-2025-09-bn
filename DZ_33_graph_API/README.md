★★★☆☆

### Цель задания
Получить опыт работы с OpenGL.

### Критерии успеха
1. Создано приложение, создающее OpenGL-окно и отображающее
в нём вращающийся с течением времени куб.
2. Bonus points за наличие текстуры на кубе.
3. Bonus points за выход из приложения по нажатию Esc.
4. Код компилируется без warning’ов с ключами компилятора
-Wall -Wextra -Wpedantic -std=c11.

Работа с OpenGL требует графическую среду.
Ниже приведенные действия проводились в WSL2.
Для доступа к графической среде WSL2 можно использовать родную среду WSLg
или же установить другую и получить к ней доступ, например, по RDP.
```
sudo apt install xfce4 xfce4-goodies -y
sudo apt install xrdp -y
```

Перед написанием кода убедитесь, что у вас установлены необходимые библиотеки.
Для установки GLFW и GLEW на Linux (Ubuntu) вы можете использовать следующие команды:
```
sudo apt-get install libglfw3-dev libglew-dev libglm-dev libsoil-dev -y
```

В задании представлены несколько программ с разными возможностями.
1. cube.c
Она создаёт разноцветный куб, который вращается вокруг своей оси по заданной траектории.

2. texture_cube.c
Это программа, демонстрирующая наложение текстур на куб в OpenGL.
Куб так же вращается вокруг своей оси по заданной траектории.

3. lesson.c
Это программа из лекции, демонстрирующая простую работу с OpenGL.
Она рисует треугольник и вращающуюся прямую линию.

Чтобы собрать любую из этих программ, можно использовать командную строку, например, для программы cube.c:
```
gcc -o cube cube.c -lGL -lGLU -lglut -Wall -Wextra -Wpedantic -std=c11
```

Но лучше запустить для сборки make.
Без параметров она выдаст ошибку:
```
make: Nothing to be done for 'all'.
```

Поэтому её нужно запускать с параметрами SRC и OUT, для любой программы, кроме cube.c и texture_cube:
```
make SRC=lesson.c OUT=lesson
```
Для  программ cube.c и texture_cube, make нужно запускать с:
```
make cube=1
```
и
```
make cube=0
```
соответственно.

Для запуска программ запустите в графической среде терминал и выполните следующие действия:
```
./cube
```
или
```
./lesson
```
или, например:
```
./texture_cube ./textures/fox.png
```
Программа texture_cube требует в качестве параметра файл с текстурами.
В каталоге с исходниками находится несколько файлов с различными текстурами.
