# ZIP Structure Parser (zipjpeg_checker)  

Простая утилита на языке C для анализа структуры ZIP-архивов. Программа сканирует файл, находит заголовки EOCD (End of Central Directory), CDFH (Central Directory File Header) и LFH (Local File Header), выводя подробную техническую информацию о каждом элементе.

## Особенности- Поиск сигнатур ZIP-формата в бинарном режиме.  
- Чтение упакованных структур (`#pragma pack(1)`) для точного соответствия спецификации.  
- Вывод метаданных: смещения, размеры сжатых/несжатых данных, контрольные суммы (CRC-32) и имена файлов.  

## Требования- Компилятор `gcc` (поддержка стандарта C11).  
- Утилита `make`.  

## СборкаЧтобы скомпилировать проект, выполните команду в терминале:  

```sh
make
```

После этого будет создан исполняемый файл zipjpeg_checker.  

## Использование  
Запустите программу, передав путь к ZIP-архиву в качестве аргумента:  
```sh
./zipjpeg_checker archive.zip
```

## Пример вывода:  
```sh
Длина файла: 15420
EOCD сигнатура найдена на смещении: 15400
Это ZIP файл

LFH сигнатура найдена на смещении: 0
Имя файла: image.jpg

Количество файлов в архиве: 1
```

## Очистка  
Для удаления объектных файлов и исполняемого файла используйте:  
```sh
make clean
```
