# Charset Converter (to UTF-8)  

Простая утилита на C для конвертации текстовых файлов из кириллических кодировок в **UTF-8**.  

## Поддерживаемые кодировкиПрограмма распознает следующие форматы (регистр не важен):   
- **CP1251**: `cp1251`, `windows-1251`  
- **KOI8-R**: `koi8-r`, `koi8r`  
- **ISO-8859-5**: `iso-8859-5`, `iso8859-5`  

## СборкаДля компиляции вам понадобится `gcc` и `make`. В корневой директории проекта выполните:  

```bash
make
```

После этого в папке появится исполняемый файл converter.  

## Использование   
Запуск программы осуществляется с тремя аргументами:  

```sh
./convert <входной_файл> <кодировка> <выходной_файл>
```

## Примеры:  

   1. Конвертация из Windows-1251:
   
   ./convert legacy_file.txt cp1251 utf8_result.txt
   
   2. Конвертация из KOI8-R:
   
   ./converter mail_text.txt koi8-r output.txt
   
   
## Очистка   
Чтобы удалить объектные файлы и исполняемый файл, используйте:  

```sh
make clean
```

## Особенности реализации  

* Программа использует таблицу перекодировки (массив), чтобы сопоставить символы расширенного ASCII с 
двухбайтовыми последовательностями UTF-8.  
* Символы с кодом меньше 128 (стандартный ASCII) копируются без изменений.
