# Simple C11 Tetris  
Классическая игра Тетрис, написанная на языке **C** с использованием библиотеки **SDL2**.  
Проект выполнен в рамках учебной практики с соблюдением строгих стандартов компиляции.  

## Особенности  
- **Gameplay**: Полноценный цикл игры с падением фигур, очисткой линий и предпросмотром следующей фигуры.  
- **Меню**: Ввод имени игрока перед началом сессии.  
- **Таблица лидеров**: Сохранение рекордов в файл `scores.txt` и вывод истории игр в консоль.  
- **Соответствие стандартам**: Код компилируется без предупреждений с флагами `-Wall -Wextra -Wpedantic -std=c11`.  

## Управление  
- **Стрелка Вверх**: Поворот фигуры.  
- **Стрелки Влево / Вправо**: Перемещение фигуры.  
- **Стрелка Вниз**: Ускоренное падение.  
- **Enter**: Начать игру (в меню).  
- **Backspace**: Удаление символа при вводе имени.  

## Требования
Для сборки необходима библиотека **SDL2**.  
```sh
sudo apt install libsdl2-dev
```

## Сборка и запуск  
1. Скомпилировать проект:  
```bash
make
```

2. Запустить игру:  
```sh
make run
```

3. Очистить временные файлы:  
```sh
make clean
```

## Файлы проекта
* tetris.c — основной исходный код.  
* Makefile — скрипт автоматизации сборки.  
* scores.txt — файл базы данных рекордов (создается автоматически).
