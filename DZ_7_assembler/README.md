## Цель задания
Консольное приложение, выводит при запуске тот же текст, что и предоставленная в листинге main.asm программа.  
Приложение работает с той же структурой данных, что и оригинальная программа.  
Код компилируется без предупреждений с ключами компилятора -Wall -Wextra -Wpedantic -std=c11.  
Функция f - итеративная версия: фильтрация через "указатель на указатель", функция m - оптимизирована: итеративный цикл вместо рекурсии.  
Устранены утечки памяти под структуры данных в ассемблерном коде.  

### Сборка проекта:  
```sh
make
```

### Запуск программ:  
```sh
./asm-prog
```

```sh
./c-prog
```

### Очистка проекта:  
```sh
make clean
```
